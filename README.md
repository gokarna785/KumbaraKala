# KumbaraKala 🏺

KumbaraKala is an Android application designed to promote traditional clay products using modern digital tools.

## 📱 Features
- Product catalog with multiple images
- Image slider (ViewPager2)
- Digital Story Card generation
- WhatsApp sharing feature
- Clean and user-friendly UI

## 🛠️ Technologies Used
- Kotlin
- Android Studio
- XML
- RecyclerView
- ViewPager2

## 🎯 Problem Solved
Traditional potters lack digital presence and struggle to market their eco-friendly products. This app helps showcase and promote clay products effectively.
